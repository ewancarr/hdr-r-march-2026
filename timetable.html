Timetable

Day 1: Beginner

This first session provides a gentle introduction to the R language and core syntax, designed for those new to the language. You will explore what makes R a widely-used tool for data analysis and how it compares to alternatives such as Python, Stata, and SPSS.

The day covers essential building blocks of the R language, including using the console, working with packages, writing scripts, and accessing help resources. You will learn about data types, handling missing values, and factors, along with techniques for subsetting data and basic control flow structures. Practical exercises will reinforce these core concepts, providing a strong foundation for subsequent sessions.

  1. Welcome and Introduction to R

    • Overview of R as a programming language.
    • How R compares to other languages like Python, Stata or SPSS.
    • Introduction to R packages, the console, writing scripts, and how to get help (?, help(), online resources).

  2. Data Types and Objects in R

    • Understanding different data types (numeric, character, logical, etc.).
    • Handling missing values (NA) in datasets.
    • Introduction to factors and their role in handling categorical data.

  3. Subsetting Data

    • Techniques for subsetting vectors, matrices, and data frames.
    • Using indices and logical conditions to select data.

  4. Control Flow (Introduction)

    • Basic control flow structures in R: if, else, and conditional logic.

  5. Practical

    • Hands-on exercises covering data types, subsetting, and basic control flow.

Day 2: Intermediate

Day 2 builds on the first session by focusing on essential data manipulation techniques, including importing/exporting data, selecting and sorting, and merging datasets using the tidyverse. You’ll also learn how to write custom functions and apply control flow to make your code more adaptable. Practical exercises will give you hands-on experience to strengthen your understanding of these key concepts and improve your coding workflow.

  1. Functions in R

    • Writing functions: How to create your own functions in R.
    • What is a function? Understanding function arguments, return values, and scope.
    • Practical: Write simple functions to perform basic tasks.

  2. Control Flow

    • Using control flow structures like if, else, ifelse(), switch().
    • Writing code that adapts based on different conditions or inputs.

  3. Data Manipulation in R

    • Overview of different data manipulation approaches: Base R, tidyverse, and data.table.
    • Using pipes (|>) for cleaner, more readable code.
    • Importing and exporting data (e.g., CSV, Excel, Stata).
    • Essential data manipulation tasks:
      • Selecting and sorting columns/rows.
      • Creating or modifying columns.
      • Merging datasets.
    • Practical: Manipulate a dataset using dplyr, pipes, and base R techniques.

Day 3: Intermediate

Day 3 builds on previous skills by focusing on more advanced data manipulation techniques, including reshaping, grouping, and summarising data using tidyverse tools. You will also be introduced to data visualisation with ggplot2, learning how to create and customise plots. The session concludes with a mini project, where participants will apply their skills to real-world tasks like data import, cleaning, manipulation, and visualisation. Practical exercises throughout the session will and help solidify your understanding.

  1. Further Data Manipulation in R

    • Reshaping data using functions like pivot_longer() and pivot_wider() from tidyverse.
    • Grouping and summarising data with group_by() and summarise() in dplyr.
    • Practical: Apply reshaping, grouping, and summarising techniques to a dataset.

  2. Plotting with ggplot2

    • Introduction to ggplot2 for creating basic plots (scatterplots, bar charts, line plots).
    • Customising plots: Adding titles, labels, themes, and modifying aesthetics (e.g., color, size).
    • Practical: Create and customise plots based on a dataset.

  3. Mini Project

    • Provide a small dataset and allow participants to apply everything they’ve learned so far.
    • Tasks include data import, cleaning, manipulation, visualisation, and basic reporting of findings.

Day 4: Advanced

Day 4 covers advanced R techniques, focusing on iteration, vectorisation, and efficient modelling. You will learn how to write efficient loops and use vectorised operations for faster computations, as well as explore purrr for more flexible iteration and broom to tidy model outputs. The session also includes working with dates and times using lubridate, and manipulating strings with stringr. You’ll also be introduced to creating dynamic reports with Quarto.

  1. Iteration and Vectorisation

    • Writing efficient loops in R (for, while, repeat).
    • The benefits of vectorised operations vs loops.
    • Using lapply for iterative tasks.
    • Introduction to purrr for more flexible and readable iteration over lists and data frames.
    • Practical: Apply these techniques to solve an iterative data manipulation or analysis task.

  2. Efficient Modelling with purrr and broom

    1. Introduction to the lm and glm functions for basic regression models.
    2. Running these models on a single dataset to extract and interpret key outputs.
    3. Using purrr to apply these models across multiple datasets or subsets (e.g., by group).
    4. Extracting and tidying model results with broom to make them easier to interpret or report (e.g., using tidy(), glance(), and augment() from the broom package).
    • Practical: Model a dataset using lm or glm, iterate over groups, and extract results.

  3. Working with Dates and Times

    • Managing and manipulating date/time data using the lubridate package.
    • Parsing, formatting, and extracting date components (e.g., year, month).

  4. String Manipulation with stringr

    • Searching, extracting, and replacing strings with stringr.
    • Regular expressions for pattern matching in strings.
    • Practical: Clean and manipulate text data using stringr and regular expressions.

  5. Dynamic Reporting with Quarto

    • Creating dynamic reports with Quarto, including embedding code and plots.

  6. Other Tips and Tricks

    • Using the conflicted package to handle namespace conflicts and ensure you are using the correct function when packages overlap.
